# C-Play Version 2.3

This is the landing page for the documentation of C-Play Version 2.3, from the application itself. Currently, this is the latest version, so the guide updates continuously for this release applies.

This update introduces a 3D view for previewing dome and sphere content, a timeline system for animating slide layers with keyframes, interactive QR code-driven presentation capabilities etc.

#### 3D View
A new visualization mode that renders layers in a 3D scene with an orbit camera, useful for previewing how content will look when projected onto domes or spheres. Toggle it from the header taskbar and/or configure it to open at startup. Optional dome overflow masking with adjustable opacity. See [Views](../guides/playback/views).

#### Playlist overrides
Per-item title, start/end time, stereo mode, grid mode, and audio file overrides saved in *.cplaylist* files. See [Build playlists](../guides/media/cplaylist).

#### Slide Timeline
Each slide in a presentation can now have its own timeline for animating layer properties over time using keyframes. 
See more in [Build presentation](../guides/media/cplaypres).

#### QR Code-Driven Presentations
Real-time QR/barcode detection in NDI and Stream layers enabling interactive immersive presentations. See [Presentation](../guides/playback/presentation).

#### Control Layers
A new type of layer called "Control" can be added that triggers operations/action in other parts of C-Play, such as Play or Load from the playlist.
See more in [Build presentation](../guides/media/cplaypres).

#### REST Commands Editor and REST Layer
C-Play v2.3 adds a REST Commands Editor for creating reusable HTTP GET/POST commands with editable name/value parameters, and a REST Layer that can trigger those commands from presentations. See [HTTP Web API](../guides/remote/api#rest-commands-editor) and [Build presentation](../guides/media/cplaypres).
