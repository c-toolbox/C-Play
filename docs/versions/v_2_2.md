# C-Play Version 2.2 

This is the landing page for the documentation of C-Play Version 2.2, from the application itself.

This version introduced the latest SGCT backend for up-to-date warping support.

In this update, C-Play uses the latest SGCT backend for up-to-date warping support, but XML configs are now dropped. Switch to JSON, as described in the [SGCT Docs](https://sgct.github.io).
