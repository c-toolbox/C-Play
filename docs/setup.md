# Setup C-Play

While C-Play is designed to be a media player used for clustered environments, there is no built-in program within C-Play to launch C-Play on different machines in a synchronized manner.

However, we have an additional system for that called [C-Troll](https://github.com/c-toolbox/C-Troll), which can launch any program on multiple computers with various configuration options.

Thus, C-Play can be launched on multiple machines with various configurations by using C-Troll as the start/stop utility.
