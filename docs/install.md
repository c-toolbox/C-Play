# Installation instructions

Please download the latest release from [GitHub](https://github.com/c-toolbox/C-Play/releases).

From 2.0 and onwards there is an installer and/or zip available for Windows.
Please install that in the exact same paths on all systems, master and clients.

To launch the application on master+nodes, we normally use our own application called [C-Troll](https://github.com/c-toolbox/C-Troll).

It's recommended that you first go through the [setup guide](setup), as well as learn how C-Play [handles and structures media](media).

When that is done, make sure you understand the application-specific [settings](settings), for instance, specifying the [locations](guides/settings/location) for C-Play to save and look for *cplaylists* and *cplayfiles*.

Hopefully the [playback](playback) features should be clear after this.
